== Description ==

Directory builder will add responsive grid of your favorite content from social media.

== Installation ==

Installing "Directory builder" can be done either by searching for "Directory builder" via the "Plugins > Add New" screen in your WordPress dashboard or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
